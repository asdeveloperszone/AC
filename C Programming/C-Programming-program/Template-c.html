Calculate Sum of Two Numbers

This program demonstrates basic input/output operations and arithmetic operations in C. It takes two integer values from the user, performs addition, and displays the result. This is one of the fundamental programs for beginners to understand how data flows in C programming.

Program Code

Here's the complete C program to calculate the sum of two numbers:

sum_two_numbers.c 
#include <stdio.h>

int main() {
    int num1, num2, sum;
    
    // Prompt user for first number
    printf("Enter first number: ");
    scanf("%d", &num1);
    
    // Prompt user for second number
    printf("Enter second number: ");
    scanf("%d", &num2);
    
    // Calculate sum
    sum = num1 + num2;
    
    // Display result
    printf("Sum = %d\n", sum);
    
    return 0;
}

How It Works

  1. Include Header: The program starts by including stdio.h which provides input/output functions like printf() and scanf().
  2. Variable Declaration: Three integer variables (num1, num2, sum) are declared to store the input numbers and their sum.
  3. First Input: The program prompts the user to enter the first number using printf() and reads it using scanf().
  4. Second Input: Similarly, it prompts for and reads the second number.
  5. Calculation: The program adds both numbers and stores the result in the sum variable.
  6. Display Result: Finally, it displays the sum using printf() with the %d format specifier.
  7. Program Termination: The program returns 0 to indicate successful execution.

Detailed Explanation

scanf() Function: The scanf() function reads formatted input from the standard input (keyboard). The "&" operator is used to pass the memory address of the variable where the input should be stored.

Format Specifiers: The %d format specifier is used for integer values. When used with printf(), it displays an integer; with scanf(), it reads an integer from input.

Arithmetic Operation: The addition operator (+) is used to add two numbers. C supports all basic arithmetic operations: +, -, *, /, and % (modulus).

  • int num1, num2, sum; - Declares three integer variables in a single statement
  • scanf("%d", &num1); - Reads an integer and stores it at the memory address of num1
  • sum = num1 + num2; - Performs addition and assigns the result to sum
  • printf("Sum = %d\n", sum); - Displays the result with a newline character
