Calculate Sum of Two Numbers

A simple program that takes two integers as input from the user and displays their sum. This exercise demonstrates basic input/output operations and arithmetic operations in C.

sum_two_numbers.c 
#include <stdio.h>

int main() {
    int num1, num2, sum;
    
    printf("Enter first number: ");
    scanf("%d", &num1);
    
    printf("Enter second number: ");
    scanf("%d", &num2);
    
    sum = num1 + num2;
    
    printf("Sum = %d\n", sum);
    
    return 0;
}

Swap Two Numbers

This program swaps the values of two variables using a temporary variable. It's a fundamental concept that helps understand variable assignment and memory manipulation.

swap_numbers.c 
#include <stdio.h>

int main() {
    int a, b, temp;
    
    printf("Enter first number: ");
    scanf("%d", &a);
    
    printf("Enter second number: ");
    scanf("%d", &b);
    
    printf("\nBefore swapping: a = %d, b = %d\n", a, b);
    
    temp = a;
    a = b;
    b = temp;
    
    printf("After swapping: a = %d, b = %d\n", a, b);
    
    return 0;
}

Check Even or Odd Number

A program that checks whether a given number is even or odd using the modulus operator. This exercise helps understand conditional statements and arithmetic operators.

even_odd.c 
#include <stdio.h>

int main() {
    int num;
    
    printf("Enter a number: ");
    scanf("%d", &num);
    
    if (num % 2 == 0) {
        printf("%d is an even number.\n", num);
    } else {
        printf("%d is an odd number.\n", num);
    }
    
    return 0;
}
