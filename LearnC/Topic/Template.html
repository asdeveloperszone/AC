Introduction to C Programming

C is a powerful general-purpose programming language that forms the foundation of modern computing. Created by Dennis Ritchie at Bell Labs in 1972, C provides low-level access to memory, efficient execution, and a simple set of keywords. It's the backbone of operating systems, embedded systems, and system software.

Use Cases

Operating Systems (Linux, Windows), Embedded Systems, System Software, Compilers, Database Management Systems, Game Engines, IoT Devices, and High-Performance Applications.

Basic Syntax

Every C program follows a basic structure with preprocessing directives, the main function, and statements. Here's the fundamental syntax:

basic_syntax.c 
#include <stdio.h>

// Main function - Entry point of program
int main() {
    // Your code here
    printf("Hello, World!\n");
    return 0;
}

Code Explanation:

  • #include <stdio.h> - Preprocessor directive that includes standard input/output library
  • int main() - The main function where program execution begins, returns an integer
  • printf() - Function to print output to the console
  • return 0 - Indicates successful program termination

Complete Example Program

Here's a complete C program demonstrating variables, input/output, and basic operations:

introduction_example.c 
#include <stdio.h>

/* Simple program to add two numbers */
int main() {
    // Variable declarations
    int num1, num2, sum;
    
    // Display prompt to user
    printf("Enter first number: ");
    scanf("%d", &num1);
    
    printf("Enter second number: ");
    scanf("%d", &num2);
    
    // Perform addition
    sum = num1 + num2;
    
    // Display result
    printf("Sum = %d\n", sum);
    
    return 0;
}

Program Breakdown:

  • Variable Declaration: int num1, num2, sum - declares three integer variables
  • scanf(): Reads input from user, &num1 provides memory address to store input
  • %d: Format specifier for integer values
  • Arithmetic Operation: sum = num1 + num2 performs addition
  • Output: printf displays the result with formatted output
